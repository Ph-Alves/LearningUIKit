# LearningUIKit

Repositório de estudos para aprender **AutoLayout com UIKit** no iOS, explorando as diferentes formas de construir layouts programaticamente e via Interface Builder.

## Sobre

Projeto prático focado em entender como o AutoLayout funciona no UIKit, cobrindo desde os conceitos básicos de constraints até layouts mais complexos com Stack Views e ancoras.

## Tópicos abordados

- [ ] NSLayoutConstraint — constraints manuais
- [ ] NSLayoutAnchor — API moderna de constraints
- [ ] UIStackView — layouts lineares horizontal e vertical
- [ ] Intrinsic Content Size e Content Hugging / Compression Resistance
- [ ] Safe Area e Layout Margins
- [ ] Scroll Views com AutoLayout
- [ ] Interface Builder vs. código

## Tecnologias

- Swift
- UIKit
- Xcode

## Como rodar

1. Clone o repositório
2. Abra o `.xcodeproj` ou `.xcworkspace` no Xcode
3. Selecione um simulador ou dispositivo
4. `Cmd + R` para rodar
